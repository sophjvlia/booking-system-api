API Documentation

Register User

POST /signup

Register a new user with email and password.

Login User

POST /login

Login an existing user with email and password.

List All Movies

GET /movies

Retrieve a list of all movies.

List Movie Details

GET /movies/:movie_id

Retrieve details of a specific movie by its ID, including available dates.

List Available Timeslots and Seats

GET /movies/:movie_id/availability/:date

Retrieve available timeslots and seats for a movie on a specific date.

Create Booking

POST /add-booking

Create a new booking for a movie timeslot.

Edit Booking

POST /edit-booking/:booking_id

Edit an existing booking by its ID.

Delete Booking

POST /delete-booking/:booking_id

Delete a booking by its ID.

List User Bookings

GET /bookings

List all bookings for the authenticated user.
